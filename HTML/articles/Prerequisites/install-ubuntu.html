Preparation and installation of Ubuntu

Foreword

So the time has come, time to install Ubuntu, the operating system I decided the group should use. The operating system(OS) is the most important and essential software we need on our computers. Different reasons for my decision of OS include, but is not limited to:

  • Ubuntu works the exact same way no matter your hardware, this allows us as a team to co-operate easier than we would be able to in other operating systems like Windows 10 or MacOS
  • Ubuntu is “windows-like”. The fact that most, if not all, of us come from a background of using windows it makes it a lot easier to use.
  • The vast amount of tutorials available to the public, for free.
  • Ubuntu features an amazing user interface nearly perfectly streamlined for effectivity.
  • The ease to work with files, directories and software in CLI's (Command Line Interface)
  • You can update all packages and even the OS itself without restarting your computer.
  • And finally, the biggest reason: Ubuntu takes more than 50% less resources. This is vital for our members who have limited RAM and computing power.

Step 1 - Download the installation media

We need to talk about some things before we go ahead and install Ubuntu. While Ubuntu clearly is the superior OS here we don't actually have to remove windows. Instead we'll do a so-called dualboot. A dualboot allows us to, at boot up, decide what operating system we want to use.

What you will need follow this tutorial include:

  • ~25gb of available storage on your disk
  • A USB stick with at least 4GB of storage (alternatively you can use a SD card, DVD/CD disc etc. depending on what hardware you have)
  • A live internet connection
  • About 30-40 mins of your day

First of you need to download the installation media to your USB. You can do that here. Alternatively, if you want faster downloads you can download it with a BitTorrent here

You should now have something that looks like the image above on your selected boot media(USB stick/SD card) and you're now ready to move to the next step

Step 2 - Partition your disk

What is a disk partition you ask? That's what I'm about to tell you.
A partition is basically a index of a range with registries on a disk. We'll use this index(partition) to tell the computer where to store windows data and where to store Ubuntu data.

Open the diskmanager by rightclicking the windows flag and then choose "Disk management"
Please be aware of the fact that the two following actions might take some time depending on your hardware. You can now sort the list of disks after "free space". rightclick the disk which has the most available storage space free. Then choose "shrink volume"

Now, if you have enough space, shrink the volume with 133120Mb. This is the amount I recommend for the Ubuntu installation if you're going to compile Unreal Engine 4. If not, you should get away with 51200Mb. You're now done here. Tim eto procede to next step

Step 3 - Installing Ubuntu

You have now downloaded the operating system and fixed a place for it to go. Now you're ready to actually install the OS on your computer.

When booting up with the USB plugged in you should be greeted by the screen above where you have the choise to try or install Ubuntu. Since we want to continue using Ubuntu we'll choose "Install Ubuntu"

Next up is choosing the keyboard layout. If you're from the USA you can just press continue

We'll choose minimal installation and we'll update software later. We'll also choose to install drivers to make sure that all your hardware works properly.

Here we will choose to install ubuntu alongside Windows. If you want to replace windows completely feel free to go ahead but I think the best option for most people is a dualboot.
Then you choose your timezone and user login. I'd recommend using a password that you'll remember or write it up. You tend to have to use it quite a lot. You'll be asked to reboot once the installation is done. Just go ahead and do so. Do boot into windows again you have to use the arrow keys to move down to windows and press enter while in the boot menu.
